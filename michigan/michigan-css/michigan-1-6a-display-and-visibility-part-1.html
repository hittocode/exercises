--------------Display is key to layout PT1---------------
-every element is a box - display affects the layout of neighboring elements ------------common values-------------
display: - inline: sits next to other elements - takes up "just enough" width and height
- block: forces line break in between the elements - default: take up all horizontal width and "just enough" height - rules can set height and width
-inline-block: - same as inline, but accepts height and width
- none: removed from page
