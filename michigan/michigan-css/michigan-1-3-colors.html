-----------Color Conventions------------- - Color names (blue, red, yellow, etc) work, but should be avoided - Hexadecimal is common convention. #0000FF, #FF0000, #FFFF00 (blue, red, yellow) (most common) - rgb (0,0,1) (1,0,0) (1, 1, 0) blue/red/yellow - rgba (0,0,1, .5) --------Accessibility------------- - Appropriate use of color is critical to web accessibility - Many more people are visually impaired or color blind than are legally blind ---------What is color contrast?---------- - You intuitively know when something has poor contrast - There are tools that quantify the contraste between text and its backgrouund - http://wave.webaim.org - http://webaim.org/resources/contrastchecker ----------- Don't use color alone to convey meaning---------- - Make sure you have alternate text for color important sections --------------Review------------- - Use web safe colors and accepted convention - test your site using a contrast checker
